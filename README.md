# 💰 Educador Financeiro IA

Uma aplicação web inteligente desenvolvida com **React + TypeScript + IA Generativa (Google Gemini)** que atua como um educador financeiro pessoal, analisando dados do usuário e gerando insights financeiros personalizados.

## 🧠 Sobre o projeto

O **Educador Financeiro IA** simula um consultor financeiro inteligente.  
O usuário preenche um formulário em etapas com informações sobre sua vida financeira e, ao final, a aplicação utiliza IA para gerar uma análise personalizada.

O objetivo é transformar dados simples em **insights acionáveis**, ajudando o usuário a entender melhor sua saúde financeira.

---

## ⚙️ Funcionalidades

- 📋 Formulário multi-etapas (wizard)
- 💾 Gerenciamento de estado com React Hooks
- 📊 Coleta de dados financeiros (receitas e despesas)
- 🤖 Integração com Google Gemini API
- 🧠 Análise financeira gerada por IA
- 💡 Sugestões personalizadas de melhoria
- 🖥️ Interface simples e responsiva
